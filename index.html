Launch your dev environments.
From the terminal.

A fast, cross-platform CLI that removes friction from your development workflow. Select or type a project name and you're off — no folder hopping, no manual installs.

~/projects
$ lo
# pick from the interactive list
○ my-app (Node / Next.js)
○ api-service (Go / Chi)
○ landing-page (Python / FastAPI)
● blog (Rust / Axum)

$ lo blog
# launches directly by name

Features

Everything Liftoff does, mapped to the commands you'll use.

lo <project>

One-command launch

Run any project from your workspace by name. No folder hopping, no fiddling — just lo and go.

auto-detect

Zero-config runtime detection

Recognises Node, Rust, Python, Go, and Java projects automatically. Picks the right package manager, runtime, and framework every time.

auto-install

Automatic dependency installation

Reads your lockfile, selects the correct package manager, and installs dependencies before launch — no separate step needed.

lo --pad <name>

Launchpads

Group multiple projects into a named launchpad and start them all together. Perfect for microservices or full-stack apps.

lo --watch <name>

Watch mode

Launch a project and monitor its resource usage in real time. CPU, memory, and process info — all inside the terminal.

--cross-platform

Cross-platform support

Works on Linux, macOS, and Windows. Includes Docker Compose detection and optional tmux integration.

Installation

Get Liftoff running in seconds. Pick your method.

Linux / macOS

curl -fsSL https://raw.githubusercontent.com/
  notliad/liftoff/main/install.sh | bash

Windows (PowerShell)

irm https://raw.githubusercontent.com/
  notliad/liftoff/main/install.ps1 | iex

Go Install

go install github.com/notliad/liftoff/cmd/lo@latest

Arch Linux (AUR)

yay -S liftoff

Build from source

git clone https://github.com/notliad/liftoff.git
cd liftoff
go build ./cmd/lo
